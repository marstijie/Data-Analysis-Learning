在数据分析基础部分，我将主要展示matplotlib在可视化中的应用
* 可视化图表基本类型：根据数据想侧重表达的内容，可以将图表归纳为5大类：趋势、分布、构成、比较和联系  
a.趋势:表示数据是如何随着时间而发生变化  
通过折线图或柱状图进行展示  
b.分布：关心数据的分布规律  
散点图来展示不同分布特征  
c.构成  
表示数据中每个部分占总体的比例，用饼图展示  
显示一段时间内的占比情况，用百分比堆积柱状图展示  
d.比较：数据在某维度上的比较  
通过簇形柱状图和并列子图来反映对比数据间的关系  
e.联系：用于衡量两种维度数据之间的关系  
即观察其中一类数据的大小是否随着另一类数据的大小有规律地变化。通常，这两类数据的统计单位不一致，且数值相差悬殊。  
使用双y轴叠加图来展示
